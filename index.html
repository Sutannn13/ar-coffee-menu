Panduan

Cara Menggunakan

1
Siapkan Marker Hiro
Cetak atau tampilkan marker Hiro di layar perangkat lain. Download Marker →
2
Klik Mulai AR
Tekan tombol "Mulai AR" di halaman utama untuk membuka mode kamera Augmented Reality.
3
Izinkan Kamera
Browser akan meminta izin akses kamera. Tekan "Izinkan" untuk melanjutkan.
4
Arahkan ke Marker
Arahkan kamera perangkat ke marker Hiro dengan jarak 15–40 cm.
5
Nikmati Coffee Cup 3D
Model 3D cangkir kopi akan muncul. Gunakan tombol kontrol untuk berinteraksi.
Tentang Project

WebAR Coffee Menu

Project ini memanfaatkan teknologi Augmented Reality berbasis web untuk menampilkan objek 3D cangkir kopi secara interaktif. Dibangun menggunakan A-Frame dan AR.js, objek 3D ditampilkan melalui marker-based tracking menggunakan marker Hiro.

Aplikasi ini cocok sebagai media demonstrasi pembelajaran mata kuliah Virtual dan Augmented Reality. Seluruh pengalaman berjalan langsung di browser tanpa memerlukan instalasi aplikasi tambahan.
